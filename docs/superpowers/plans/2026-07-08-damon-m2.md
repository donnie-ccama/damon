**Goal:** M2 — keyring-backed provider keys with `damon key` commands, OpenRouter models resolving at spawn, Codex + OpenCode runtimes with embedded-memory bridges, the session-end reflection Stop hook, plus ledgered debt (kill partial-failure, GitError::Io).

Read the actual files — do not assume M1 shapes.

**Tech:** keyring 2 (already in workspace deps), rpassword 7 (add), std::io::IsTerminal, serde_json (present).

## Global Constraints

- Plaintext keys never touch disk. Resolution order for `${keyring:<account>}`: (1) env seam `DAMON_KEY_<ACCOUNT>` (uppercase, `-`→`_`) — documented test/CI escape hatch; (2) OS keyring service `"damon"`, account `<account>`. Missing → error naming the fix: `run: damon key set <account>`.
- Generic `${VAR}` values resolve from damon's own environment (pre-staged behavior — keep).
- Bridges regenerate before every spawn; all bridge files git-excluded via returned paths (existing mechanism).
- Codex and OpenCode both read `AGENTS.md`; they share one embedded-memory generator. Claude additionally gets `.claude/settings.json` with the Stop hook.
- Reflection hook: `damon hook reflect` reads Claude Code hook JSON on stdin; if `stop_hook_active` is true → exit 0 (allow stop); else print the reflection instruction to **stderr** and exit 2 (block once). No state files.
- Tests must not touch the real OS keychain (use the env seam / nonexistent accounts); real-keychain tests are `#[ignore]`.
- Threat model note (decided): `tmux -e` key injection is kept; exposure is same-user-only and millisecond-transient on the short-lived tmux client. Document in README.
- Every task ends green (`cargo test`), then commit.

```rust
fn resolve_from_keyring(
    model_key: &str,
    name: &str,
    account: &str,
) -> anyhow::Result<(String, String)> {
    // Test/CI seam and container escape hatch: DAMON_KEY_<ACCOUNT>.
    let seam = format!(
        "DAMON_KEY_{}",
        account.to_uppercase().replace(['-', '.'], "_")
    );
    if let Ok(v) = std::env::var(&seam) {
        if !v.is_empty() {
            return Ok((name.to_string(), v));
        }
    }
    let entry = Entry::new("damon", account)
        .map_err(|e| anyhow::anyhow!("keyring unavailable for account {account:?}: {e}"))?;
    let password = entry.get_password().map_err(|_| {
        anyhow::anyhow!(
            "model {model_key:?} needs the {account:?} key for {name:?} — run: damon key set {account}"
        )
    })?;
    Ok((name.to_string(), password))
}
```

```rust
use std::io::{IsTerminal, Read};

pub fn set(provider: &str) -> anyhow::Result<()> {
    let key = if std::io::stdin().is_terminal() {
        rpassword::prompt_password(format!("key for {provider} (input hidden): "))?
    } else {
        let mut buf = String::new();
        std::io::stdin().read_to_string(&mut buf)?;
        buf
    };
    let key = key.trim();
    if key.is_empty() {
        anyhow::bail!("key must not be empty");
    }
    keyring::Entry::new("damon", provider)?.set_password(key)?;
    println!("stored key for {provider} (OS keyring, service \"damon\")");
    Ok(())
}

pub fn rm(provider: &str) -> anyhow::Result<()> {
    keyring::Entry::new("damon", provider)?.delete_password()?;
    println!("removed key for {provider}");
    Ok(())
}
```

```rust
use std::io::Read;

pub fn reflect() -> anyhow::Result<()> {
    let mut input = String::new();
    std::io::stdin().read_to_string(&mut input).ok();
    let v: serde_json::Value = serde_json::from_str(&input).unwrap_or(serde_json::Value::Null);
    if v.get("stop_hook_active").and_then(|b| b.as_bool()).unwrap_or(false) {
        return Ok(()); // second pass: reflection done, allow the stop

         MEMORY.md. Update AGENT.md, USER.md, MEMORY.md and skills/ in your canonical \
         memory directory with anything durable you learned, then finish."
    );
    std::process::exit(2);
}
```
