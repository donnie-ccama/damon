# Damon M7 — Tokyo Night theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the ratatui TUI a cohesive, accent-only Tokyo Night ("Night") color scheme via a single `theme` module that the render layer pulls from.

**Architecture:** New `crates/damon/src/tui/theme.rs` holds the palette (named `Color::Rgb` constants) and semantic style helpers returning `ratatui::style::Style`. `view.rs` is refactored to call the helpers instead of inline style literals, so the palette lives in one place. Accent-only: the terminal background is never painted. This is one cohesive task — the module and every call site land in one commit (otherwise unused helpers trip clippy's dead-code lint).

**Tech Stack:** Rust, ratatui 0.29 (`Color::Rgb`, `Style`, `Block::border_style`, `Line::styled`, `Cell`), `TestBackend` for style-assertion tests. Design source of truth: `docs/superpowers/specs/2026-07-09-damon-m7-design.md`.

## Global Constraints

- Accent-only: never set a pane/body background. The only `bg` used is the selection bar's `#292e42` on the selected row.
- Palette = Tokyo Night "Night": `BG_HIGHLIGHT #292e42`, `MUTED #565f89`, `BLUE #7aa2f7`, `CYAN #7dcfff`, `GREEN #9ece6a`, `MAGENTA #bb9af7`, `RED #f7768e`, `ORANGE #ff9e64`.
- Live session badge stays **green** (`#9ece6a`). Status line: idle hint muted; active message orange; **error messages (status starting with `"error"`) red**.
- Selection is a bar (bg `#292e42` + magenta fg + bold) that must NOT clobber a row's own span colors (green badge, red INVALID stay).
- Only these files change: new `crates/damon/src/tui/theme.rs`, `crates/damon/src/tui/view.rs`, `crates/damon/src/tui/mod.rs`.
- Final state: `cargo test -p damon` green (existing text tests unaffected + new style tests pass) and `cargo clippy -p damon --all-targets` no warnings. There is no `[lib]` target — use `cargo test -p damon`, not `--lib`.
- TDD, DRY, YAGNI, one commit.

- [ ] **Step 3: Create the theme module**

Create `crates/damon/src/tui/theme.rs`:

```rust
//! Tokyo Night ("Night") palette + semantic style helpers. Accent-only:
//! nothing here paints a pane background except the selection bar.
use ratatui::style::{Color, Modifier, Style};

const BG_HIGHLIGHT: Color = Color::Rgb(0x29, 0x2e, 0x42);
const MUTED: Color = Color::Rgb(0x56, 0x5f, 0x89);
const BLUE: Color = Color::Rgb(0x7a, 0xa2, 0xf7);
const CYAN: Color = Color::Rgb(0x7d, 0xcf, 0xff);
const GREEN: Color = Color::Rgb(0x9e, 0xce, 0x6a);
const MAGENTA: Color = Color::Rgb(0xbb, 0x9a, 0xf7);
const RED: Color = Color::Rgb(0xf7, 0x76, 0x8e);
const ORANGE: Color = Color::Rgb(0xff, 0x9e, 0x64);

/// Exposed for the view's style-assertion tests.
pub(crate) const SELECTION_BG: Color = BG_HIGHLIGHT;
pub(crate) const BORDER_FG: Color = BLUE;

pub fn border() -> Style {
    Style::default().fg(BLUE)
}
pub fn title() -> Style {
    Style::default().fg(BLUE).add_modifier(Modifier::BOLD)
}
pub fn selection() -> Style {
    Style::default()
        .bg(BG_HIGHLIGHT)
        .fg(MAGENTA)
        .add_modifier(Modifier::BOLD)
}
pub fn tab_active() -> Style {
    Style::default()
        .fg(CYAN)
        .add_modifier(Modifier::BOLD | Modifier::UNDERLINED)
}
pub fn tab_inactive() -> Style {
    Style::default().fg(MUTED)
}
pub fn team() -> Style {
    Style::default().fg(BLUE).add_modifier(Modifier::BOLD)
}
pub fn header() -> Style {
    Style::default().fg(BLUE).add_modifier(Modifier::BOLD)
}
pub fn badge() -> Style {
    Style::default().fg(GREEN)
}
pub fn model_col() -> Style {
    Style::default().fg(CYAN)
}
pub fn uptime_col() -> Style {
    Style::default().fg(MUTED)
}
pub fn invalid() -> Style {
    Style::default().fg(RED)
}
pub fn hint() -> Style {
    Style::default().fg(MUTED)
}
pub fn status_msg() -> Style {
    Style::default().fg(ORANGE)
}
pub fn status_error() -> Style {
    Style::default().fg(RED)
}
pub fn error_block() -> Style {
    Style::default().fg(RED)
}
```

Register it in `crates/damon/src/tui/mod.rs` — add after `pub mod snapshot;` (line 6):
