- `cargo test` green + `cargo clippy --all-targets` clean under the new names. No `[lib]` target in the app crate ā€” use `cargo test` (workspace) / `cargo test -p cortado`.
- Work on branch `m8-rename-cortado` (already created; the M8 spec is committed there).

- [ ] **Step 2: `git mv` the crate directories**


- [ ] **Step 3: Global case-aware replace over tracked files (excluding historical + binary + lock)**

```bash
git ls-files \
  | grep -vE '^docs/superpowers/(plans|specs)/' \
  | grep -vE '(^|/)Cargo\.lock$' \
  | grep -vE '\.(rvf|db)$' \
  | grep -vE '\.rvf\.lock$' \
  | while IFS= read -r f; do
      perl -i -pe 's/DAMON/CORTADO/g; s/Damon/Cortado/g; s/damon/cortado/g' "$f"
    done
```

This rewrites: every `Cargo.toml` (package `name`, workspace `members`, the four `path` deps in root `[workspace.dependencies]`, the `{ workspace = true }` dep keys), all `use cortado_core::`/etc. module paths, all `CORTADO_*` env vars, all string literals (socket `"cortado"`, session prefix `cortado_`, `@cortado_model`, `.cortado-tmp`, `.cortado-exclude.lock`, `# cortado` markers, config dir, `cortado hook reflect`), README, and `packaging/aur/*`. (The M8 spec/plan and the orchestrator design doc are excluded, so they keep their historical `damon` text.)

Rename the tap repo and update the formula:
```bash

```
Clone/locate the tap, then:
```bash

```
Edit `Formula/cortado.rb`: `class Damon` ā†’ `class Cortado`; `url` ā†’ `https://github.com/donnie-ccama/cortado/archive/refs/tags/v0.2.0.tar.gz`; `sha256` ā†’ `<SHA256_V020>`; `desc`/`homepage` ā†’ Cortado/new repo; the `install`/`test` blocks build+install the `cortado` binary. Then:
```bash

brew install donnie-ccama/cortado/cortado
```
