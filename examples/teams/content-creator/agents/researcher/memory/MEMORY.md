# Memory

Your notes: project conventions, tool quirks, lessons learned. Keep an
index here; put long topics in their own files next to this one.

## Write-back protocol

- Save: stated preferences, corrections, durable facts, confirmed approaches.
- Skip: trivia, one-off state, anything easily re-discovered.
- Consolidate rather than endlessly append; delete notes that turn out wrong.
- At session end, review the conversation and update memory and skills.
